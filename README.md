# APImagination - 天气查询应用

这是一个使用 React 和 OpenWeatherMap API 构建的简单天气查询应用。

## 功能特点

- 输入城市名称查询天气
- 显示温度、天气状况和湿度
- 响应式设计
- 加载状态显示
- 错误处理

## 开始使用

1. 克隆项目
2. 安装依赖：
   ```bash
   npm install
   ```
3. 在 `src/App.tsx` 中替换 `YOUR_API_KEY` 为您的 OpenWeatherMap API key
4. 启动开发服务器：
   ```bash
   npm start
   ```

## 技术栈

- React
- TypeScript
- Material-UI
- Axios
- OpenWeatherMap API

## 获取 API Key

1. 访问 [OpenWeatherMap](https://openweathermap.org/)
2. 注册账号
3. 在个人中心获取 API Key

## 注意事项

- 请确保您有稳定的网络连接
- API 调用可能有频率限制
- 城市名称请使用英文
